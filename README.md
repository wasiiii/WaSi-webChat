# 功能介绍

### 注册登录模块
1. 用户名按c语言规则
2. 手机邮箱不能重复

### 用户管理模块
1. 管理员可以查看所有人的邮箱和手机
2. 管理员可以修改除自己外所有人的邮箱和密码，手机不可更改
3. 普通用户只能修改自己的邮箱和密码，不能查看信息

### 聊天模块
1. 有未读消息红点提示
2. 发送图片
3. 即时聊天
4. 接收离线消息
